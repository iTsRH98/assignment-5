Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select elements from the DOM. Here are the most commonly used methods.

1.getElementById: Selects a specific element.
2.getElementsByClassName: Selects all elements with a specific class.
3.getElementsByTagName: Selects all elements with a specific tag name.
4.querySelector: Selects the first element that matches a CSS selector.
5.querySelectorAll: Selects all elements that match a CSS selector. etc

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML Gets or sets HTML content inside an element.

innerText Retrieves or sets only the visible text inside an element.

textContent Retrieves or sets all text content inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event triggered on a child element propagates (travels) up to its parent elements.
